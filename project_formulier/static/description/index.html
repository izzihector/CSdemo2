Every Formulier Type has a three phase.

Formulier Type One First Phase (Opportunity Stage)

In First Phase (Opportunity Phase) User can add necessary information like Opportunity Information and upload and edit images, videos and documents from frontend


Formulier Type One Second Phase (Quotation Stage)

In Second Phase (Quotation Phase) User can see information about Opportunity Phase and add Information About Quotation Question. Upload video, images and document from fronted and edit images from feontend. add Digital Signature in Quotation Stage.
